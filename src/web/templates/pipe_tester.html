หน้าทดสอบ Named Pipe

หน้านี้เปิดอ่าน named pipe (FIFO) ตาม path ที่เลือก แบบ real-time เพื่อทดสอบว่ามีข้อมูลถูกเขียนเข้ามาจริงหรือไม่ — ใช้ได้ทั้งกับ pipe ที่ VAS เขียนเอง (เช่น integration "Pipe I/O" ของ QR500) และ pipe ที่มี third-party process ภายนอกเขียนเข้ามา ไม่ผูกกับฟีเจอร์ใดฟีเจอร์หนึ่งโดยเฉพาะ

ข้อควรระวัง: named pipe ส่งข้อมูลแต่ละบรรทัดให้ "ผู้อ่านคนแรกที่รออยู่" เพียงคนเดียว ถ้ามี process อื่นเปิดอ่าน pipe เดียวกันค้างอยู่พร้อมกัน (เช่น terminal ที่ยังรัน cat ค้างไว้จากการทดสอบครั้งก่อน) ข้อมูลอาจไปถึง process นั้นแทนหน้านี้แบบสุ่ม — เช็คได้ด้วย sudo lsof /tmp/vas_qr_pipe หรือ sudo fuser -v /tmp/vas_qr_pipe ก่อนทดสอบ
